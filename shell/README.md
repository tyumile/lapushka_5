# shell

## Назначение
Shell — это общий фронтовый слой системы, который объединяет автономные модули в единую точку входа, не ломая их локальные интерфейсы.

## Что делает
- задает общий стиль интерфейса;
- дает общую навигацию;
- объединяет ссылки и точки входа в локальные UI модулей;
- формирует единое ощущение системы.

## Что не делает
- не забирает к себе бизнес-логику process-модулей;
- не переписывает локальные страницы модулей;
- не заменяет orchestrator.

## Структура папки
- `main.py` — точка входа shell
- `AGENTS.md` — локальная политика
- `README.md` — описание роли и структуры
- `ui/` — общий UI-слой
- `tests/` — тесты
- `logs/` — логи
