# shared

Эта папка содержит общие соглашения и контракты проекта Lapushka 5.

## Что здесь находится
- `cabinet_contract.md` — минимальный контракт сущности кабинета и правила использования `cabinet_id`
- `ui_contract.md` — общие правила внешнего вида и поведения UI
- `task_registry.md` — правила работы с workflow-базой задач
- `task_registry.sql` — SQL-схема таблиц `tasks`, `task_handoffs`, `task_reviews`, `task_events`
- `../task` и `../scripts/task.py` — CLI для чтения своей очереди (`mine`), просмотра карточки (`show`), истории (`list`) и записи результатов (`add`)

## Назначение
`shared` не должен превращаться в место, куда складывают бизнес-логику модулей. Здесь должны лежать только общие соглашения, контракты и легкие повторно используемые описания.
